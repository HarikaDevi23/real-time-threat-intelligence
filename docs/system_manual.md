# System Manual
## Threat Intelligence Dashboard

The dashboard displays real-time data about detected threats. It includes:
- Live list of detected attacks
- Risk scoring for each threat
- Response actions executed

---

## Automated Mitigation Overview

The system auto-detects and mitigates threats based on:
- AI prediction (via GPT-4)
- IP blocking (iptables)
- Contextual responses (WAF, 2FA, etc.)
